# Journal Skeletons — fill-in-the-blank for every phase

Per-phase pre-filled templates so you don't burn minutes staring at a blank page. Copy the skeleton into `journal/phase_N_*.md` at the start of each phase, fill the blanks as you run the phase, commit the final version.

**Rubric reminder** (each entry scored 0/2/4 on):

- D1 Harm framing (who bears what cost, in $)
- D2 Metric → cost linkage (the metric tied to money)
- D3 Trade-off honesty (what was sacrificed, quantified)
- D4 Constraint classification (hard/soft + penalty in $)
- D5 Reversal condition (signal + threshold + duration)

Target ≥ 3.0 average to pass. Entries that fail any dimension by being vague (`"if data changes"`, `"good metric"`, `"maybe"`) score 0 on that dimension.
