# Phase 9 — Codify (pointer)

Phase 9 (Codify) does NOT run inline with the per-sprint walk. It runs at the close of the workshop via `/codify` — see [`workflow-08-codify.md`](./workflow-08-codify.md).

Why the split: per-sprint codification fragments the lessons. Holding `/codify` to the end means the lessons have ALL the journal entries to learn from, not a partial set.

The Phase 9 journal file produced is `journal/phase_9_codify.md`. The running lessons file is `playbook/appendix-a-lessons.md`.

**Next:** open [`workflow-08-codify.md`](./workflow-08-codify.md) at the close of the workshop.
